Вызов удаленной функции

Модификация программного кода классов DemoFunction, DemoAdapter, DemoApplication и JSP для реализации удаленного вызова методов функций не требуется.

Изменения вносятся только в конфигурационные файлы приложений на локальном и удаленном компьютере. При использовании протокола HTTP для вызова удаленной функции на удаленном компьютере необходимо модифицировать и контекстный файл web.xml.
