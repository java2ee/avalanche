Вызов удаленной функции (HTTP)

Для обеспечения удаленного вызова методов функции с использованием протокола HTTP необходимо скопировать демонстрационное приложение на удаленный компьютер и внести изменения в конфигурацию обеих копий приложений (удаленную и локальную).

На удаленном компьютере необходимо добавить в конфигурационный файл секцию <connector> cо значением атрибута class "HTTPConnector" и вложенной секцией <publish>, в которой указывается публикуемая в коннекторе функция.

В локальный конфигурационный файл необходимо добавить секцию <interface> и изменить значение атрибута uri в секции <адаптер>. Секцию с локальной функцией можно исключить из конфигурации.
