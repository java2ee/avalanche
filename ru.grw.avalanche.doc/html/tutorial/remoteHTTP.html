Вызов удаленной функции, протокол RMI

Для обеспечения удаленного вызова методов функции необходимо на удаленном компьютере добавить в конфигурационный файл секцию <connector> cо значением атрибута class "RMIConnector" и вложенной секцией <publish>.

В локальный конфигурационный файл необходимо добавить секцию <interface> и изменить значение атрибута uri в соответствующей секции <адаптер>.
